### Via Help Center Button[​](#via-help-center-button "Direct link to Via Help Center Button")

1. Click "Help Center" in the navigation menu
2. A new tab opens with comprehensive documentation

### Using Search Help Function[​](#using-search-help-function "Direct link to Using Search Help Function")

1. Click the "Search Help" button (magnifying glass icon)
2. Type your query into the search bar
3. Results appear as you type

## Navigating the Documentation Site[​](#navigating-the-documentation-site "Direct link to Navigating the Documentation Site")

- Browse categories for specific topics
- Use the search function for targeted information
- Access video tutorials and guides
