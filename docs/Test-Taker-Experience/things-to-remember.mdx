title: "Things to Remember"
description: "Neurapulse uses speed and accuracy to measure risk. For optimal performance, please make sure your mobile device is prepared for the test by following these guidelines."
---

## Screen Protector[​](#screen-protector "Direct link to Screen Protector")

If you have a screen protector, please note:

* Floating screen covers (built into phone case) should be removed
* Adhesive screen protectors do not need to be removed

## Cleanliness[​](#cleanliness "Direct link to Cleanliness")

Your screen should be clean and dry.

## Stylus[​](#stylus "Direct link to Stylus")

The test is designed to be taken using your index finger (thumb is not recommended). A stylus may be used if assistive tools are required.
